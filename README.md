## 🛠️ 프로젝트 생성

### 📋 사전 준비물
- **Java 17 이상** 설치
- IDE: **IntelliJ** 또는 **Eclipse** 설치

### 🏗️ 스프링 프로젝트 생성 방법
1. [Spring Initializr](https://start.spring.io)로 이동하여 스프링 프로젝트를 생성합니다.
2. 아래 설정을 선택합니다:

   - **Project**: Gradle - Groovy
   - **Spring Boot**: 3.x.x
   - **Language**: Java
   - **Packaging**: Jar
   - **Java**: 17 또는 21

3. **Project Metadata**
   - **Group**: `hello`
   - **Artifact**: `core`

4. **Dependencies**
   - 선택하지 않습니다.

> ⚠️ **주의**: 스프링 부트 3.0 이상 및 JDK 17 이상을 반드시 사용해야 합니다.

이 프로젝트에서는 아래의 중요한 스프링 개념들을 다룹니다:

- 빈 생명주기와 스코프 관리
